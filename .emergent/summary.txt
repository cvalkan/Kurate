**Key Technical Concepts**
-   **Topic Diversity Hypothesis**: The core finding of the session. SI wins when papers are topically diverse; PW wins when papers are topically homogeneous. Ground Truth (GT) type is a proxy for this, not the cause.
-   **Precomputed Caching**: The main performance strategy. Expensive DB computations are run once, stored in version-controlled JSON files (), and loaded into an in-memory cache on server startup. This made validation pages >100x faster.
-   **Non-blocking Async Tasks**: The long-running rating generation was made non-blocking by switching from concurrent batches to a sequential loop with  between each call, yielding the event loop and keeping the UI responsive.
