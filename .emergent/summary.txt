**Key Technical Concepts**
-   **Goal-Directed Matchmaking**: The system was refactored to prioritize matches that directly contribute to convergence goals (CI reduction, top-K cross-matches), making it much more efficient.
-   **Calibration Ratio**: A new tunable parameter (0-100%) that controls the probability of matching a new, unranked paper against another new paper (for fast CI reduction) vs. an established, converged paper (for fast score calibration against the existing ranking).
-   **Tier-Based Validation**: Using the final acceptance decisions from peer review (Oral, Spotlight, Poster, Reject) as a ground truth to validate the AI tournament's ranking. This is a stronger signal than using noisy, uncalibrated reviewer scores.
-   **Asynchronous Data Ingestion**: Refactoring the OpenReview import into a background task to handle long-running operations without causing server timeouts.
-   **Statistical Artifact Correction**: Fixing a small-sample-size artifact in the convergence charts by including all papers (even those with 0 matches) in the correlation calculation and using a deterministic tie-breaker (paper title hash) for identical scores.
